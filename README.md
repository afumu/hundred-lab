# 百创计划

这是我的百创计划总母仓库。

我会在这个仓库里持续沉淀五类正式资产：

- 100 个项目
- 100 个技能
- 100 个产品
- 100 篇文章
- 100 个视频

这些内容都和 AI、工程化、自动化、内容生产、个人效率和技术实践有关。它们既会在这个仓库里保留完整内容，也可能在外部继续拆分成独立的公开仓库。

## 这个仓库的定位

这个仓库不是单一项目，而是整个百创计划的总母仓库。

它承担五个职责：

1. 作为百创计划的总入口
2. 作为原始 Idea 的统一暂存区
3. 作为所有内容的统一沉淀区
4. 作为 Idea 和正式条目的组织中心
5. 作为后续拆分独立仓库的母本

## 默认工作方式

后续我会把日常工作中的上下文、经验、问题、流程、代码、想法和复盘不断沉淀到这里。

在这个仓库里，默认会优先做这些事情：

1. 先把值得留下的原始想法放进 `ideas/`
2. 再判断一段上下文是否有机会长成产品
3. 再拆解它是否适合做成项目、技能、文章和视频
4. 默认优先面向 AI 从业者，其次是开发者
5. 默认优先沉淀个人效率方向的产品和工具
6. 默认按公开开源的方式组织内容

## 当前仓库结构

当前仓库分为几层：

- `ideas/`：原始想法池
- `content/`：正式内容区
- `site/`：百创计划展示站点
- `docs/`：规则与计划文档
- `templates/`：标准模板
- `tools/`：常见工具最佳实践
- `assets/`：公共素材
- `scripts/`：自动化脚本
- `registry/`：全局索引和台账
- `archive/`：归档内容

## 目标

这个计划的目标不是简单堆数量，而是尽可能把真实工作中产生的经验，持续转化成：

- 可复用的产品
- 可开源的项目
- 可沉淀的方法
- 可传播的文章
- 可表达的视频

在正式转化之前，原始想法先进入 `ideas/`，作为长期积累和后续升级的输入池。
