# 在外面远程控制 Codex，终于爽起来了

这篇文章记录一套移动端远程控制 Codex 的个人工作流：

- 用 YepAnywhere 管理和控制 Codex
- 用 Cockpit Tools 管理和切换 Codex Plus 账号
- 用 Codex 执行软件层面的工作
- 在移动端远程发起任务、推进修改、沉淀内容

这不是一篇工具测评，而是一篇工作流体验记录。重点是：当 Codex 可以被远程稳定控制，账号又能被统一管理之后，移动端使用 AI 编程工具的体验会发生很大变化。
