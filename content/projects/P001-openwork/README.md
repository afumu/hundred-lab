# OpenWork

OpenWork 是百创计划记录的第一个开源项目。

源码仓库：

- <https://github.com/afumu/openwork>

## 项目定位

OpenWork 是一个面向组织部署的浏览器智能体平台。它把用户工作区、管理后台、产品网关和按用户隔离的运行时组合在一起，让团队可以部署一套 Web-first 的 AI workspace。

这个目录只作为百创计划中的项目说明入口，不复制源码。后续 OpenWork 的代码仍然在独立 GitHub 仓库中持续更新。
