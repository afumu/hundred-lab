# 文章条目模板

## 目录建议

```text
Axxx-article-name/
├── README.md
├── article.md
├── meta.yaml
├── outline.md
├── references.md
└── assets/
```

## article.md 建议结构

1. 开头问题
2. 背景和上下文
3. 方法或过程
4. 关键判断
5. 结果与复盘
6. 可复用结论

## meta.yaml 建议字段

```yaml
id: A001
title: 文章标题
slug: article-name
status: candidate
visibility: public
tags: []
related_projects: []
related_products: []
related_skills: []
related_videos: []
```
