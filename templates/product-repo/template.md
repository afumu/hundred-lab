# 产品条目模板

## 目录建议

```text
PRxxx-product-name/
├── README.md
├── meta.yaml
├── spec.md
├── roadmap.md
├── docs/
├── app/
├── landing/
└── assets/
```

## README 建议结构

1. 产品定位
2. 目标用户
3. 核心问题
4. 最小可交付版本
5. 当前状态
6. 后续路线

## meta.yaml 建议字段

```yaml
id: PR001
title: 产品名称
slug: product-name
status: candidate
visibility: public
tags: []
repo: null
related_projects: []
related_skills: []
related_articles: []
related_videos: []
```
