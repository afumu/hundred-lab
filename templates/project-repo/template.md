# 项目条目模板

## 目录建议

```text
Pxxx-project-name/
├── README.md
├── meta.yaml
├── src/
├── tests/
├── docs/
└── assets/
```

## README 建议结构

1. 项目是什么
2. 解决什么问题
3. 谁会使用
4. 如何运行
5. 当前状态
6. 关联文章和视频

## meta.yaml 建议字段

```yaml
id: P001
title: 项目名称
slug: project-name
status: candidate
visibility: public
tags: []
repo: null
related_products: []
related_articles: []
related_videos: []
```
