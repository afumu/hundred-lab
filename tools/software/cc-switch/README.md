# CC Switch

## 当前定位

CC Switch 是一个跨平台的 AI 编程工具管理软件。

GitHub：

- <https://github.com/farion1231/cc-switch>

它不是只服务 Codex，而是用于统一管理 Claude Code、Codex、Gemini CLI、OpenCode 和 OpenClaw 等多个 CLI 工具。

## 核心价值

CC Switch 的价值在于把多个 AI 编程工具的配置、供应商、账号和相关能力集中管理。

它更适合作为“多 AI 编程工具管理中心”，而 Cockpit Tools 更偏向当前本机的 Codex 账号管理和切换。

## 主要能力

根据 GitHub 仓库说明，CC Switch 主要支持：

- Claude Code、Codex、Gemini CLI、OpenCode、OpenClaw 多工具管理
- 供应商配置管理
- 一键切换供应商
- 系统托盘快速切换
- MCP 管理
- Prompts 管理
- Skills 管理
- 会话管理
- 使用量和成本追踪
- 跨平台桌面应用，支持 Windows、macOS、Linux

## 推荐使用场景

- 同时使用多个 AI 编程 CLI 工具
- 需要统一管理 Claude Code、Codex、Gemini 等工具配置
- 需要快速切换不同 API 供应商
- 需要统一维护 MCP、Prompt 和 Skill
- 团队或个人有多套 AI 编程工具链

## 和当前工具栈的关系

当前仓库的软件工作默认以 Codex 为主。

在这个前提下：

- Codex：默认执行工具
- YepAnywhere：控制 Codex 的工具
- Cockpit Tools：管理和切换 Codex Plus 账号
- 豆包输入法：快速语音输入任务和上下文
- CC Switch：作为多 AI 编程工具统一管理方案的备选或补充
