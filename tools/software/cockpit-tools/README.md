# Cockpit Tools

## 当前定位

Cockpit Tools 是本机已安装的软件工具，用于管理和切换 Codex 相关账号。

本机检测到的信息：

- 桌面应用：`/usr/share/applications/Cockpit Tools.desktop`
- 启动命令：`cockpit-tools`
- 可执行文件：`/usr/bin/cockpit-tools`
- URL Scheme：`cockpit-tools`、`cockpittools`

用户原始口述里提到的 `COCPkit TOOL`，实际应记录为本机安装的 `Cockpit Tools`。

## 核心价值

Cockpit Tools 的核心价值是方便管理大量 Codex Plus 账号，并支持快速切换账号。

它适合解决的问题：

- 多个 Codex Plus 账号需要统一管理
- 高频使用 Codex 时，需要快速切换可用账号
- 手动退出、登录、切换账号太低效
- 大批量账号需要更清晰的管理入口
- 软件工作默认交给 Codex 后，需要账号层面的基础设施

## 推荐使用场景

- Codex Plus 多账号管理
- Codex 账号一键切换
- 高频 Codex 工作中的账号调度
- 多账号额度耗尽后的快速切换
- 搭配 YepAnywhere 控制 Codex
- 搭配豆包输入法做口述输入，再交给 Codex 执行

## 和 Codex 工作流的关系

Codex 是默认的软件工作执行工具。

Cockpit Tools 负责 Codex 账号层面的管理和切换，属于 Codex 工作流的基础设施。

推荐组合：

1. 用豆包输入法快速口述任务。
2. 用 YepAnywhere 控制 Codex。
3. 用 Codex 执行软件层面的工作。
4. 用 Cockpit Tools 管理和切换大量 Codex Plus 账号。

## 注意事项

- Codex 账号和 token 属于敏感信息，应优先本地管理。
- 使用账号切换工具前，应确认当前 Codex 客户端的登录状态和配置文件位置。
- 批量账号管理需要注意账号合规、权限边界和长期维护成本。
- 如果后续找到 Cockpit Tools 的官方 GitHub 或发布页，应补充来源链接和安装说明。
