# 豆包输入法

## 当前定位

豆包输入法是日常输入和口述记录工具。

它的语音输入识别非常准，适合用来把临时想法、工作上下文、工具使用经验和长段口述快速转成文字。

## 推荐使用场景

- 快速记录想法
- 口述工作上下文
- 输入长段中文内容
- 辅助整理百创计划中的 Idea
- 在使用 Codex 时，快速把任务要求说出来
- 写文章、脚本、复盘和工具最佳实践初稿

## 当前最佳实践

- 先用语音快速说出完整想法，不要一边说一边过度编辑。
- 语音输入完成后，再交给 Codex 做结构化整理、资产拆解和目录沉淀。
- 对于百创计划相关内容，可以先口述原始上下文，再让 Codex 判断是否进入 `ideas/` 或升级为正式资产。
- 对于工具使用经验，可以先用豆包输入法记录真实操作，再整理到 `tools/` 对应目录。

## 和 Codex 的配合方式

豆包输入法负责高效输入，Codex 负责整理、判断和落盘。

推荐流程：

1. 用豆包输入法口述原始想法或任务。
2. 把口述内容交给 Codex。
3. Codex 判断内容应该进入 `ideas/`、`tools/`，还是正式资产目录。
4. Codex 按仓库规则整理成结构化文档。
