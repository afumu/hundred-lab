# Skills 工具箱

## 当前定位

这里用于记录我本地常用的 skills。

这些 skills 不是普通软件，而是我给 AI 工作流安装的一层“可调用能力”。它们让 Codex、Claude Code 或其他 agent 在特定场景下，直接进入更成熟的流程，而不是每次都从零开始。

对当前工具栈来说，skills 的价值很直接：

- 把高频任务变成可复用能力
- 把隐性的工作经验变成显性的操作入口
- 让内容生产、AI 编程、浏览器操作、研究和自动化都更稳定
- 让我的个人方法论逐步沉淀成长期资产
