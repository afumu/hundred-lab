# agent-browser

## 当前定位

`agent-browser` 是浏览器自动化基础设施。

它不是某一个网站专用的 skill，而是很多网页操作能力的底座。只要 agent 需要打开网页、点击按钮、截图、抓文本、登录或做页面测试，基本都可以从它开始。

## 核心能力

- 打开网页并保持浏览器会话
- 获取页面元素快照和可交互元素引用
- 执行点击、输入、选择、滚动和等待
- 提取页面文字、标题和 URL
- 截图和页面状态检查

## 适用场景

- 自动化操作网站
- 测试网页应用
- 远程替用户完成网页流程
- 在浏览器里抓取结构化信息
- 作为其他平台技能的底层能力

## 在当前工具栈里的价值

它相当于“网页世界的机械手”。后面像 B 站操作、公众号发布、平台自动化、浏览器插件验证，很多都要建立在它上面。
