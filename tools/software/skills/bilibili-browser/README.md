# bilibili-browser

## 当前定位

`bilibili-browser` 是建立在 `agent-browser` 之上的 B 站专项操作 skill。

它把“怎么在 B 站里打开页面、复用登录态、浏览视频、点击视频、抓取文本”这些常见动作封装成了更稳定的流程。

## 核心能力

- 打开 B 站主页或指定页面
- 复用 B 站认证状态
- 浏览视频列表和用户空间
- 点击视频并继续操作
- 抓取页面上的文字信息

## 适用场景

- 打开 B 站做内容浏览
- 抓取 B 站页面资料
- 为后续视频学习或素材整理做预处理
- 让 agent 帮忙操作 B 站页面

## 在当前工具栈里的价值

如果你后面做 B 站收藏分析、视频学习系统、自动化内容整理，这个 skill 就是你在 B 站上的专用入口。
