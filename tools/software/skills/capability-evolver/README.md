# capability-evolver

## 当前定位

`capability-evolver` 是一个偏元能力的 self-evolution skill。

它关注的不是完成某个业务任务，而是让 agent 回看自己的历史、识别低效点或失败点，再尝试更新自身能力和记忆结构。

## 核心能力

- 分析运行历史和错误模式
- 发现可演进的能力缺口
- 支持自修复和自进化
- 以更系统化的方式积累 agent 的演进记录

## 适用场景

- 想让 agent 自动发现自身可以改进的地方
- 做长期自进化实验
- 搭建更完整的进化型 agent 系统

## 在当前工具栈里的价值

你已经在思考“自净化”“自进化”“学到东西后自动沉淀”，这个 skill 正好对应这种方向。它更偏实验性和系统性能力，而不是日常小工具。
