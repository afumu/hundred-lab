# coremi

## 当前定位

`coremi` 是一个专门用于执行 Coremi AI 新闻编辑部流程的业务型 skill。

它不是通用 agent 工作流，而是一套围绕 AI 新闻与观点生产的九步流，把采集、判断、分析、写作和编辑都拆成了稳定步骤。

## 核心能力

- 执行 Coremi 固定 9 步新闻工作流
- 在本地生成结构化 Markdown 过程产物
- 组织新闻抓取、观点整理、竞争分析和写作编辑
- 适合把资讯处理成完整内容流程

## 适用场景

- 跑 AI 新闻编辑部工作流
- 生成新闻写作过程文件
- 做资讯型内容生产
- 为 Coremi 产品验证实际工作流

## 在当前工具栈里的价值

它是 Coremi 产品方法论的一个“工作流投影”。也就是说，它不是单点小工具，而是一个已经开始承接真实业务链路的专项 skill。
