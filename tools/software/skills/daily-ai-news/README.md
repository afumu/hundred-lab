# daily-ai-news

## 当前定位

`daily-ai-news` 是用来生成每天 AI 新闻简报的 skill。

它的核心不只是搜新闻，而是把抓取、过滤、分类、摘要和链接整理成一套稳定流程，减少自己手工看大量零碎资讯的负担。

## 核心能力

- 聚合多个 AI 新闻来源
- 结合网页搜索发现近期更新
- 过滤重复信息和弱信号内容
- 按主题分类整理
- 输出带来源链接的简报

## 适用场景

- 每天看 AI 新动态
- 做 AI 行业信息跟踪
- 作为写文章、做选题、做投资或产品判断前的输入
- 给自己的资讯系统提供原始情报

## 在当前工具栈里的价值

它很适合接到你后面“行业监听”“需求监听”“专家观点降噪”那一类产品想法里，作为最上游的信息获取能力。
