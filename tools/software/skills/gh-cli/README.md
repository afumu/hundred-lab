# gh-cli

## 当前定位

`gh-cli` 是 GitHub CLI 的知识和操作参考 skill。

它解决的是“怎么从命令行稳定地操作 GitHub”，包括仓库、Issue、PR、Actions、认证和发布等一整套命令面。

## 核心能力

- GitHub 登录与账号切换
- 仓库和 PR 操作
- Actions 与 CI 查看
- Release、Gist、组织、项目等 GitHub 能力说明
- 命令行 GitHub 工作流参考

## 适用场景

- 想从终端操作 GitHub
- 做 PR、Issue、CI 处理
- 配置 `gh auth` 和 git 集成
- 把 GitHub 操作接入自动化流程

## 在当前工具栈里的价值

对你现在这种高频整理仓库、提交改动、同步 GitHub 的工作方式来说，它很实用，能让 GitHub 相关动作更稳定、少踩坑。
