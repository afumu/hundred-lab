# html-ppt

## 当前定位

`html-ppt` 是我现在最适合拿来快速做高质量演示页面和 HTML 幻灯片的 skill。

它的重点不是传统办公软件式 PPT，而是直接生成静态 HTML 演示稿，可以有主题、布局、动画、演讲者模式和更强的设计表达。

## 核心能力

- 快速生成 HTML 幻灯片
- 提供多套主题、版式和动画
- 支持演讲者模式和逐字稿
- 适合技术分享、产品介绍、汇报和小红书图文式展示
- 产物天然适合发布、预览和继续改造

## 适用场景

- 做技术分享稿
- 做产品介绍页
- 做课程模块讲解
- 把文章或笔记转成一套可展示的演示内容
- 需要一个比传统 PPT 更自由、更好看的表达方式

## 在当前工具栈里的价值

这个 skill 很适合百创计划，因为仓库里本身就会持续产出文章、产品、项目和方法论。`html-ppt` 能把这些内容直接变成可演示、可发布的视觉表达层。
