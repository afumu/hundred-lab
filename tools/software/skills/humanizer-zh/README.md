# humanizer-zh

## 当前定位

`humanizer-zh` 是一层中文文本去 AI 味的编辑能力。

它不是简单润色，而是专门识别那些很像 AI 写作的模式，再把文本改得更自然、更有人味、更像真实的人在说话。

## 核心能力

- 识别 AI 常见文风痕迹
- 去掉宣传腔、模板腔和空泛表达
- 保留原意的前提下重写文本
- 增加真实语气、观点感和节奏变化

## 适用场景

- 改文章
- 改公众号稿
- 改口播文案
- 改产品说明和介绍页
- 让 AI 生成内容更适合公开发布

## 在当前工具栈里的价值

对你这种高频用 AI 写东西的人来说，这个 skill 很重要。它能把“能写出来”进一步推进到“能发出去，而且不显得像机器写的”。
