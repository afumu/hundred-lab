# jiangxiang

## 当前定位

`jiangxiang` 是把表达讲得更香、更有价值感、更容易转化的 skill。

它不是普通文案润色，而是围绕“卖点、动力、阻力、定位、场景、故事、数字、情绪”等维度，去重新组织一段表达，让它更容易打动人。

## 核心能力

- 诊断一段表达为什么不香
- 提炼真正的卖点
- 把抽象内容讲得更具体、更有画面
- 帮助产品、课程、服务、活动或 IP 提升价值表达
- 适合反复陪练和多轮升级

## 适用场景

- 改课程介绍
- 改产品文案
- 改直播稿、短视频口播
- 做销售表达和定位表达
- 想提升内容的转化感和说服力

## 在当前工具栈里的价值

它和 `humanizer-zh` 不一样。`humanizer-zh` 更像去 AI 味编辑器，`jiangxiang` 更像价值表达增强器。后面做产品介绍、项目介绍、文章和视频脚本时都能用。
