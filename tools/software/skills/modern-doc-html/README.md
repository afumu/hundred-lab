# modern-doc-html

## 当前定位

`modern-doc-html` 是把文档或文章转换成现代化 HTML 页面的一层能力。

它适合把原本只是 Markdown 或说明文档的内容，快速升级成更好看、更适合展示和分享的网页。

## 核心能力

- 把文档转换成现代深色主题 HTML
- 支持渐变、卡片布局和动画
- 适合长文档分段生成
- 有明确的页面结构和配色模板

## 适用场景

- 把技术文档做成网页
- 把文章做成更适合传播的展示页
- 做演示风格的内容页面
- 给方法论、课程、说明稿做更有设计感的 HTML 外壳

## 在当前工具栈里的价值

它和 `html-ppt` 很像，但更偏单页或文档页。后面百创计划里的文章、方法论、课程说明、产品介绍页都可以用它快速做视觉升级。
