# notebooklm-py

## 当前定位

`notebooklm-py` 是 Google NotebookLM 的程序化操作入口。

它的价值不只是“能用 NotebookLM”，而是可以用命令和自动化工作流去创建 notebook、添加资料、生成内容和下载结果，让 NotebookLM 变成一个可被 agent 调度的知识处理工具。

## 核心能力

- 创建 NotebookLM notebook
- 批量添加多种来源的资料
- 和资料内容对话
- 生成播客、总结和其他 NotebookLM 产物
- 下载和管理结果

## 适用场景

- 围绕一个主题建立知识库
- 把书、视频、网页、PDF 等统一喂给 NotebookLM
- 自动化做主题研究
- 把 NotebookLM 接入自己的学习和内容工作流

## 在当前工具栈里的价值

它把 NotebookLM 从“网页工具”升级成“可编排能力”。对你后面的读书产品、知识处理系统、学习 agent 都很关键。
