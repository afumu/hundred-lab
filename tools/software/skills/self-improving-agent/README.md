# self-improving-agent

## 当前定位

`self-improving-agent` 是用来记录 learnings、errors 和 feature requests 的持续改进 skill。

它和 `capability-evolver` 的区别在于：`capability-evolver` 更像主动进化引擎，`self-improving-agent` 更像稳定的经验沉淀机制。

## 核心能力

- 记录失败、纠正和经验
- 把 learnings 分类沉淀到 Markdown 文件
- 为后续改进提供结构化输入
- 让 agent 的成长不只停留在当前上下文里

## 适用场景

- 命令失败后记录教训
- 用户纠正之后记录改进点
- 发现更优做法时沉淀最佳实践
- 建长期可追踪的能力改进日志

## 在当前工具栈里的价值

这个 skill 非常适合接到你后面要做的 `coding-harness`、经验库、自进化系统里。它是“把失败变资产”的基础设施。
