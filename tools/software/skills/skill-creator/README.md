# skill-creator

## 当前定位

`skill-creator` 是用来创建和更新 skill 的基础能力。

它解决的问题不是“写一段说明”，而是如何把一个经常重复出现的能力，沉淀成结构清晰、可触发、可复用的 skill 目录。

## 核心能力

- 说明 skill 应该解决什么问题
- 定义 skill 什么时候触发
- 解释 `SKILL.md` 应该怎么写
- 说明 skill 目录里可以放哪些脚本、参考资料和资源
- 帮助把零散经验整理成真正可安装、可维护的 skill

## 适用场景

- 想把某个工作流升级成 skill
- 想给 Codex / Claude Code 增加新能力
- 想修改已有 skill 的触发条件和结构
- 想把方法论、工具调用规则、脚本和参考资料打包起来

## 在当前工具栈里的价值

这个 skill 很重要，因为百创计划后面本身就会持续产出新 skills。它相当于“造 skill 的 skill”，是整个技能体系不断长大的基础能力。
