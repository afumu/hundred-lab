# videocut-install

## 当前定位

`videocut-install` 是视频剪辑 skills 的环境准备入口。

它不直接产出视频内容，但负责把 Node.js、FFmpeg、火山引擎语音识别等依赖安装好，让后面的字幕、口播识别和剪辑链路能跑起来。

## 核心能力

- 安装视频处理依赖
- 配置 API Key
- 验证环境是否可用
- 明确整个视频处理体系的前置条件

## 适用场景

- 第一次搭建视频工作流
- 新电脑初始化
- 视频处理脚本跑不起来时排查环境

## 在当前工具栈里的价值

它属于“前置基建型 skill”。没有它，后面的口播识别、字幕生成和视频自动处理就很难稳定运行。
