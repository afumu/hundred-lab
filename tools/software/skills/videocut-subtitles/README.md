# videocut-subtitles

## 当前定位

`videocut-subtitles` 是字幕生成与烧录 skill。

它把“提取音频 -> 语音转录 -> 词典纠错 -> 人工审核 -> 烧录字幕”串成一套完整流程，适合把口播或视频素材快速变成带字幕的视频。

## 核心能力

- 从视频中提取音频
- 调用语音识别生成字幕
- 用词典提升专有名词识别准确率
- 让 agent 做一轮自动校对
- 支持人工审核后再烧录

## 适用场景

- 给口播视频加字幕
- 给讲解视频加字幕
- 需要较规范字幕流程的视频内容处理

## 在当前工具栈里的价值

它是视频发布工作流里最容易复用的一层。后面不管是做课程、做分享视频、做短视频，这个 skill 都是基础设施。
