# videocut-trim-talking-head

## 当前定位

`videocut-trim-talking-head` 是面向口播视频的转录与口误识别 skill。

它的重点不是直接做最终剪辑，而是先把口播视频拆成可审查、可删除、可复核的结构化结果，帮助用户更高效地完成后续剪辑。

## 核心能力

- 提取音频并上传
- 调用语音识别做转录
- 生成字级别字幕
- 识别口误、静音和可删除片段
- 输出审查网页和删除任务清单

## 适用场景

- 剪口播视频
- 清理口误和停顿
- 为后续精剪做预处理
- 让用户先审核，再决定怎么删

## 在当前工具栈里的价值

它很适合你的内容生产方式，因为你经常会有口述、分享、录屏和讲解类内容。这个 skill 能把最费时间的初筛工作先自动化。
