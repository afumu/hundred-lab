# yt-dlp

## 当前定位

`yt-dlp` 是一个跨平台视频下载能力。

它负责把 YouTube、X、TikTok、Instagram 等平台的视频下载下来，或者只抽音频，为后续学习、剪辑和分析做准备。

## 核心能力

- 下载多平台视频
- 提取视频音频
- 选择清晰度和格式
- 获取视频基本信息
- 作为其他视频学习和剪辑流程的前置步骤

## 适用场景

- 下载线上视频做学习
- 提取音频做转录
- 为字幕和剪辑流程准备素材
- 为视频内容分析或知识提取准备输入

## 在当前工具栈里的价值

这是视频工作流里的基础搬运层。后面的 `视频学习`、`剪口播`、`字幕` 等流程，很多都可以把它当成前置资料准备工具。
