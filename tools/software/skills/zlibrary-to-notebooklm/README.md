# zlibrary-to-notebooklm

## 当前定位

`zlibrary-to-notebooklm` 是一个非常明确的学习链路型 skill：从 Z-Library 下载书籍，再自动送进 NotebookLM。

它把“找书 -> 下载 -> 上传 -> 建知识库”这一串步骤打通了，适合拿来快速建立围绕一本书的对话式阅读环境。

## 核心能力

- 从 Z-Library 链接提取书籍信息
- 自动下载书籍文件
- 优先处理 PDF，必要时降级 EPUB
- 自动创建 NotebookLM notebook
- 把书上传进去，变成可继续对话和总结的知识源

## 适用场景

- 想快速围绕一本书建立 NotebookLM 知识库
- 做主题学习时要批量处理书籍
- 想把阅读流程从手工上传改成自动化

## 在当前工具栈里的价值

它是学习系统里的一个“资料接入器”。后面你如果做读书产品、知识图谱、主题学习 agent，这个 skill 都是很关键的底层入口。
