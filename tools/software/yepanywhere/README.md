# YepAnywhere

## 当前定位

YepAnywhere 是用于远程管理和控制 Codex 的软件工具。

在当前软件工作流里，它承担的是控制入口角色：用户可以在外面、在移动端，把任务交给电脑上的 Codex 执行。

## 核心价值

YepAnywhere 的核心价值是让 Codex 不再只能坐在电脑前使用。

它让移动端变成 Codex 的远程控制入口：

- 在外面也能发起 Codex 任务
- 移动端可以直接控制软件工作流
- 想到任务时可以马上交给 Codex
- 降低从灵感到执行之间的延迟
- 让 Codex 更像一个随时可调度的远程执行器

## 推荐使用场景

- 外出时远程控制 Codex
- 移动端发起仓库整理任务
- 远程让 Codex 修改文档或代码
- 把临时想法直接交给 Codex 落盘
- 和豆包输入法配合，把口述内容变成 Codex 任务
- 和 Cockpit Tools 配合，支撑 Codex 多账号工作流

## 和当前工具栈的关系

- 豆包输入法：负责语音输入
- YepAnywhere：负责远程控制 Codex
- Cockpit Tools：负责 Codex Plus 多账号管理
- Codex：负责实际执行软件任务
